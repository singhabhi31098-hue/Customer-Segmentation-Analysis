# Customer Segmentation Analysis

## Project Overview
This project analyzes customer purchasing behavior and groups customers into different segments based on spending patterns and transaction data.

## Tools Used
- Microsoft Excel
- Microsoft Power BI
- Python

## Key Features
- Data cleaning and preparation
- Customer segmentation based on purchasing behavior
- Interactive dashboard to visualize customer groups
- Insights to support targeted marketing strategies

## Insights
The analysis identifies high-value customers, frequent buyers, and low-engagement customers. These insights help businesses design better marketing strategies and improve customer retention.
